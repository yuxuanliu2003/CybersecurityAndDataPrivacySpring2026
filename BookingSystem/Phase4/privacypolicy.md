# Privacy Policy

## Introduction
This booking system processes personal data of registered users in order to provide booking services.

## Data Collected
The system collects the following personal data:
- Email address (username)
- Password (stored as a secure hash)
- Birthdate
- Booking information
- Login timestamps and IP addresses

## Purpose of Processing
Personal data is used for:
- User authentication
- Reservation management
- System security and logging

## Legal Basis
Processing of personal data is based on the performance of a contract and legitimate interest in maintaining system security.

## Data Security
Passwords are stored using secure hashing (bcrypt) and authentication logs are maintained for security monitoring.

## Data Subject Rights
Users have the right to:
- Access their personal data
- Request correction
- Request deletion where applicable

## Contact
For questions related to personal data processing, users should contact the system administrator.
